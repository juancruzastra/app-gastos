# App Gastos

Aplicación simple en HTML, CSS y JavaScript para cargar gastos e ingresos desde el celular.

## Funciones

- Pantalla de agregar
- Pantalla de resumen
- Gastos e ingresos
- Categorías frecuentes y modal de más categorías
- Medio de pago por íconos
- Historial editable
- Filtro por mes, tipo, medio de pago y orden
- Exportación a Excel
- Guardado local en el navegador

## Publicación

Subir estos archivos a la raíz de GitHub Pages:

- index.html
- style.css
- script.js
